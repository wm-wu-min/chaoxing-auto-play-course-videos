📺 超星学习通单课视频无限循环助手 (Tampermonkey)
🚀 专为超星学习通打造的单节视频循环播放脚本，去除复杂功能，界面简洁，只做一件事：自动刷单课时长。

📖 项目简介
本脚本是一个专为大学生设计的超星学习通（Chaoxing）辅助工具。它能自动检测当前页面的视频任务，模拟真实播放，并自动向服务器上报进度。
其最大特色是「无限循环」功能：开启后，当一个视频播放完成后，脚本会自动刷新页面，无限次地重刷当前单节视频，实现“刷时长”的效果。

⚠️ 免责声明：本脚本仅供学习交流使用，请勿利用其进行违反学校规定的刷课行为。使用本脚本产生的一切后果由使用者自行承担。

✨ 功能特点
✅ 单视频无限循环：一键开启，播完自动刷新重播，解放双手。

✅ 倍速播放支持：内置倍速调节器，可自由调整模拟播放的速度（默认 1 倍）。

✅ 可视化悬浮面板：在网页右侧提供清爽的操控面板，实时显示运行状态和进度日志。

✅ 进度安全上报：模拟真人播放间隔（每30秒上报一次），降低被系统检测的风险。

✅ 即开即用：无需复杂配置，安装油猴脚本后进入视频页面即可看到控制面板。

🔧 安装与使用
前置条件
需要在浏览器中安装 Tampermonkey (油猴) 插件。

Chrome 商店安装

Edge 商店安装

安装脚本
点击本项目仓库中的 chaoxing-auto-play.user.js 文件（或你所存放的脚本文件名）。

点击文件内容右上角的 Raw 按钮。

Tampermonkey 将自动识别并弹出安装界面，点击 安装 即可。

使用方法
打开超星学习通（Chaoxing），进入包含视频任务的课程页面。

脚本会自动加载，在页面右上角会出现一个悬浮面板。

手动播放视频（务必让视频开始播放，建议静音并切到后台）。

在面板中勾选 「开启无限重刷本课」 开关。

视频播放完毕后，脚本会提示“准备执行循环”，并自动刷新页面重新开始下一轮。

💡 提示：建议将面板中的 「播放倍速」 调至 1.5 或 2.0，可以更快速地完成单次循环。

位置：页面右上角。

包含：循环开关、倍速输入框、运行日志窗口。

🛠️ 技术实现
语言：原生 JavaScript (ES5+)

请求库：基于 GM_xmlhttpRequest 跨域 API。

加密算法：使用 Blueimp MD5 库模拟超星标准盐值加密（enc 特征码生成）。

核心逻辑：通过 setInterval 定时器模拟播放秒数，以固定的间隔向 reportUrl 发送进度请求。

⚠️ 常见问题 (FAQ)
Q：为什么开启了无限循环，视频播完没有刷新？
A：请检查浏览器是否开启了“自动播放”限制，或者当前页面的视频是否已经被系统的任务进度点标记为“已完成”。如果是，请手动刷新页面再次加载。

Q：脚本上报失败，显示跨域错误怎么办？
A：这是浏览器的安全策略限制，通常不影响脚本执行。请确保你是从超星学习通主域名进入的页面，并且油猴插件拥有跨域权限（默认是允许的）。

Q：我可以修改代码让它支持别的平台吗？
A：完全可以。目前代码针对超星学习通封装，支持 chaoxing.com、edu.cn 等常见域名，如果你有其他需求，欢迎 Fork 本项目进行二次开发。
